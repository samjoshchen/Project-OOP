# MartMinds
##  Features

### Customer Features
- **User Authentication** - Register and login with secure credentials
- **Product Browsing** - View available products with detailed information
- **Shopping Cart** - Add multiple items and checkout
- **Order Management** - Create orders and track order history
- **Payment Processing** - Multiple payment methods (Cash, E-Wallet, Credit Card)
- **Balance Management** - Top-up and manage account balance
- **Profile Management** - View and update user profile

### Admin Features
- **Product Management** - Add, update, delete, and view products
- **Order Monitoring** - View all customer orders
- **User Management** - View all registered users
- **Inventory Control** - Manage product stock levels

### System Features
- **Session Management** - Secure login sessions with role-based access
- **Exception Handling** - Custom exceptions for payment, orders, and stock
- **Data Validation** - Input validation for emails, phone numbers, credit cards
- **Singleton Pattern** - Efficient service management
- **MVC Architecture** - Clean separation of concerns

## How to Run (In Terminal)

1. **Compile the project:**
   ```bash
   javac -d bin src/com/martminds/**/*.java
   ```

2. **Run the application:**
   ```bash
   java -cp bin com.martminds.Main
   ```
